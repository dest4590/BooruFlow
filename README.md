# BooruFlow

### A simple, fast image browser for booru-style sources

</div>

What it is

- A small, fast UI for browsing and filtering images from booru-style sources.
- Built with Vite and Vue 3. It focuses on straightforward image browsing: search, grid view, lightbox, and per-image controls.

Why this exists

- I made this to avoid bloated gallery apps. It handles the common tasks well: quick search, easy filtering, and simple image controls — nothing fancy, just useful.

Quick features

- Grid-based image browsing with lightbox preview
- Search bar and filter UI
- Basic image controls and settings panel
- Mobile-friendly layout and keyboard navigation

Getting started (user)

Just navigate to: https://dest4590.github.io/booruflow/

Getting started (developer)

1. Install dependencies

```bash
npm install
```

2. Run the dev server

```bash
npm run dev
```

3. Open the app

- Visit the address shown by Vite: http://localhost:5173

Build for production

```bash
npm run build
```
